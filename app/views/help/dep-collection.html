Data Returns – guidance

How to format Environment Agency data

From:
Environment Agency
First published:
19 January 2016

Follow rules to format data that meets Environment Agency standards, then submit your data returns online.

Contents

  1. Environmental Permitting Regulations (EPR) data rules
  2. General rules: time and symbols
  3. Agreed lists: parameters and CAS, units and measures, reference periods and methods

You can submit landfill emissions compliance data using the online service. First you must correctly format data to meet Environment Agency standards. Once you’ve formatted your data you must save it as a CSV file.

Set up your data management systems using these rules to produce an automated export of data in the correct format.

Follow these steps to correctly format your data:

  • apply rules to correctly format data under Environmental Permitting Regulations (EPR)
  • check if there are any further rules that apply to specific return types
  • check if you need to apply general rules on allowable formats, eg for symbols
  • save your data file as CSV
