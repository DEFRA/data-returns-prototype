Return Type Rules

Return Types are derived from the old monitoring return forms at the back of your permit. They have been prefixed with the name of the monitoring regime e.g. Environmental Permitting Regulations or Industrial Emissions Directive (EPR/IED) or Pollution Inventory (PI). Each Return Type will have specific formatting rules depending on the type of information. Some of them will be very specific e.g. Landfill Gas, while others will span several sectors such as Surface Water or Air Emissions.
