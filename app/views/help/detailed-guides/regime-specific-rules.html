Data Returns – guidance

Data rules for EPR compliance monitoring

From:
Environment Agency
First published:
12 December 2015
Last updated:
13 January 2016, see all updates
Part of:
Data Exchange Protocol

These are the rules what data should be provided under each field heading, and how it should be presented.

Field Long name Rules
EA_ID Core Permit reference without any separators, slashes or variation numbers. For EPR permits this should be 2 letters four numbers and 2 letters and for older Waste Management Licences it should be a 5 or 6 figure number Yes
Return Type This must be off the controlled list of Return Types. Yes
Monitoring Date/Time (Start) The date and time (seconds are optional) of the spot sample. If monitoring is for a period of time, it is normal practice to provide the date at the end of the monitoring period, however, please see Regime specific rules. Yes
Substance Chemical substance or other parameter from the substances list Yes
Numerical Value The measurement Yes
Unit Emissions readings should be provided in %v/v or mg/m3. Temperature in DegC and Pressure in KPa or mBar Yes
Monitoring Period Name The monitoring period covered by the submission e.g. Q1 2014 Only for returns that cover more than one point in time i.e. are not spot samples
Reference Period State the reference period required by your permit e.g. 15 min average Must be provided where available
Test Methodology/Standard e.g. BS EN1111 where relevant Must be provided where available
Sample Ref The company may provide its own reference in this box or leave blank Optional
Sampled By Leave blank unless there is a specific requirement to provide details e.g. make and model, of the monitoring equipment Optional
Comments Free text Optional
Site Name Optional crosscheck on permit number Optional
CiC In nearly all cases this should be set to 'No' or left blank as these emissions measurements are public register information under EU and UK legislation. Optional
