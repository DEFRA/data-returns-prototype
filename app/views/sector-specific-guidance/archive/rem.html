REM compliance monitoring data rules

Field short heading Description and rules Example
EA_ID Enter the 8-character permit reference. This should be 2 letters, 4 numbers and 2 letters, without any prefixes or suffixes, eg EPR/ or /V002.You must complete this field in all data returns. You can send data returns for multiple permits. Make sure your file is no larger than around 2MB. AB1234ZZ
Site_Name Enter the site name for your installation as given in your permit.Only use letters and numbers. Limit the character length to 255. Mycity Power Station 
Rtn_Period Enter the return year in a 4-digit format (YYYY). 2016

Mon_Point Enter ‘Site’. Site[d]
Mon_Date Leave blank.

Qualifier Leave blank.  
Ref_Period Leave blank.  
Value Enter numerical value.[f] 4.5
Txt_Value Leave blank.  
Unit You must give the unit or measure used in your data return. Select an entry from the unit or measure controlled list.[g] [DEP1200] m3
Meth_Stand Leave blank.  
Comments Use[h] this optional field for free-text comments about the row of data - eg, if you were unable to take a reading[i]. You can also use this field if you wish to give your own system references to store with the results.Avoid unnecessary[j] commas and symbols. You can enter a maximum of 255 characters.

REM data returns controlled lists

You need to select the correct name from these controlled lists under certain field headings, such as parameter, when you complete your data returns. These lists support Environment Agency data standards and help reduce errors in your compliance monitoring.

You can download all standard data terms from the Environment Agency controlled lists for these field headings:

  1. Unit or measure

  2. Parameter (substance name)

  3. Reference period

  4. Return period

  5. Monitoring standard or method

  6. Return type

  7. Text value

  8. Qualifier

  9. Waste codes

  10. D or R codes
