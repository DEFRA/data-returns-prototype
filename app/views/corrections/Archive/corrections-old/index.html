2 Mon_Date "30/02/2015"

You must give a monitoring date. This must be formatted as:

  • DD-MM-YYYY or YYYY-MM-DD
  • only use hyphens as separators

Monitoring time is optional according to return type, but should be formatted as hh:mm:ss
